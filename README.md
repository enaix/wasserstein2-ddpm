# Wasserstein-2 DDPM

Experiments for the paper "Wasserstein-2 Generative Networks" https://arxiv.org/abs/1909.13082 on the Summer2Winter dataset.

## Notebooks

- [W2GN_authors.ipynb](notebooks/W2GN_authors.ipynb) contains the original implementation based on the code provided by the authors.
- [W2GN.ipynb](notebooks/W2GN.ipynb) is an implementation based on the paper, needs to be fixed.
- [W2GN_Diff.ipynb](notebooks/W2GN_Diff.ipynb) is an attempt to train the OT as a denoising diffusion model.
